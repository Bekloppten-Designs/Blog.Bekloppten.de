Homeserver Software installieren mit Docker

🐳 Was ist Docker?

Docker ist eine Software, mit der Programme in Containern ausgeführt werden können.
Ein Container enthält alles, was eine Anwendung braucht: Code, Bibliotheken und Einstellungen.
Dadurch läuft die Anwendung auf verschiedenen Geräten nahezu identisch.

Mit Docker Compose kannst du mehrere Container gemeinsam definieren und starten.
So bleibt die Konfiguration übersichtlich und leicht reproduzierbar.

Docker wird verwendet häufig für:

☁️ Homeserver
🌐 Webanwendungen
🧪 Testumgebungen
🚀 Deployment und Automatisierung

🛠️ 1. Docker installieren

⚠️ Bitte überprüfe Befehle vor dem Ausführen. Ich übernehme keine Haftung. 


sudo apt update
sudo apt install -y ca-certificates curl gnupg

sudo install -m 0755 -d /etc/apt/keyrings

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update
sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

Testen

Teste einmal, ob die Installation erfolgreich war:


docker --version

Du solltest so eine Ähnliche Ausgabe bekommen:


Docker version 29.5.3, build d1c06ef

Jetzt bist du bereit, Software wie Cloudflared, Nextcloud oder Nginx zu Installieren
